# 📊 Data Analytics Project: Customer Behavior Analysis

## 📌 Overview

This project focuses on analyzing customer behavior using data analytics techniques. It involves data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard creation using Power BI. The final output includes insights, a structured report, and a presentation for business decision-making.

---

## 📁 Dataset

* Source: Customer behavior dataset (CSV format)
* Contains details such as:

  * Customer ID
  * Age, Gender
  * Purchase Amount
  * Product Category
  * Region
  * Subscription Status
  * Shipping Type

---

## 🛠️ Tools & Technologies Used

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Dashboard creation & visualization)
* **Excel** (basic validation & reporting)
* **Jupyter Notebook**
* **PowerPoint (PPT)** for presentation

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Checked data structure, null values, and data types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column names
* Fixed data type issues

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Studied purchase patterns
* Identified top products and revenue trends
* Visualized insights using charts

### 4. SQL Analysis

* Loaded cleaned data into PostgreSQL / MySQL / SQL Server
* Performed queries such as:

  * Total revenue by category
  * Customer segmentation
  * Average spending analysis
  * Subscription-based insights

### 5. Power BI Dashboard

* Created interactive dashboard including:

  * KPI cards (Total Revenue, Total Customers, Avg Spend)
  * Bar charts and pie charts
  * Category-wise and region-wise analysis
  * Filter/slicers for dynamic exploration

### 6. Reporting & Presentation

* Prepared a structured report with insights
* Created a PowerPoint presentation summarizing:

  * Objective
  * Methodology
  * Key findings
  * Business recommendations
